STEP 7 / 9

AI 基礎模型

在數十億張自然影像上預訓練的基礎模型，能以「提示 (prompt)」方式分割顯微影像，零訓練橫跨多個術式——但原版 SAM 在生物上很少是最佳選擇。

Foundation models pretrained on billions of natural images segment microscopy via prompts — zero-shot across modalities. But raw SAM is rarely the right model for biology on its own.

觀念

一、提示式分割

傳統 DL 分割：訓練好的模型輸入影像，輸出 mask，無法互動。

SAM (Segment Anything Model, 2023) 開啟了新範式——用戶給提示 (prompt)：點一下、畫個 box、給張粗略 mask、甚至文字，模型即時回傳該物件的精確 mask。

四種主要 prompt 類型：

  • Point：點擊「我要這個物件」
  • Box：拉個 bounding box
  • Mask：粗略 mask → 精細化
  • Text：「all cells」「nuclei only」（CLIP 結合）

Traditional DL: a trained model takes an image and outputs a mask — not interactive.

SAM (Segment Anything Model, 2023) opened a new paradigm — the user provides a prompt (a point, box, rough mask, or text) and the model returns a precise mask in real time.

Four main prompt types:

  • Point: "I want this object"
  • Box: a bounding box
  • Mask: coarse mask → refined
  • Text: "all cells", "nuclei only" (CLIP-grounded)
模型生態

二、SAM 家族與生物醫學分支

Model發表特色

工作流程

三、互動式分割流程

開啟 napari + Micro-SAM

pip install micro-sam napari → napari 啟動，安裝 plugin。

pip install micro-sam napari → run napari, install the plugin.

計算 embedding

選好模型（vit_b_lm, vit_b_em, vit_h_lm…），SAM encoder 跑一次得到嵌入向量。之後所有 prompt 都即時。

Pick a model (vit_b_lm, vit_b_em, vit_h_lm…). SAM encoder runs once to compute embeddings; all prompts thereafter are real-time.

互動 prompt

切到 napari 的 point / box layer，點一下細胞 → 即時 mask；不夠精細可再點 negative point 排除背景。

Use napari's point/box layer — click a cell to get an instant mask; refine with negative points to exclude background.

自動模式

AMG = Automatic Mask Generation：在整張影像撒點，過濾低品質結果，得到所有物件 mask。可作為其他模型的 pseudo-label。

AMG = Automatic Mask Generation: scatter point prompts across the image, filter low-quality outputs to get all object masks. Useful as pseudo-labels for other models.

3D / 影片

Micro-SAM 提供 volumetric 與 tracking 模式：在 2D 標一張，跨 z 或 t 自動追蹤連結。

Micro-SAM offers volumetric & tracking modes: label one 2D slice, then auto-propagate across z or t.

⚠️
不要把 SAM 用在 3D：原版 SAM 是 2D 模型，逐切片獨立跑會造成 z 方向 ID 不一致。請改用 Micro-SAM volumetric / MedSAM2 / Cellpose 3D。 Don't run vanilla SAM on 3D: it's a 2D model; per-slice independent runs produce inconsistent IDs across z. Use Micro-SAM volumetric / MedSAM2 / Cellpose 3D instead.
Fine-tuning

四、Fine-tune 策略

策略參數量情境
Linear probe凍結 backbone，僅訓練線性層資料 < 1k 張、做下游分類Freeze backbone, train linear head only< 1k labels, downstream classification
LoRA / Adapters凍結 backbone，加 ~1% 可訓練參數中等資料，效率好Freeze backbone, add ~1% trainable paramsMedium data, efficient
Prompt encoder only凍結 image encoder，只訓 prompt + mask decoder想保留 SAM 通用能力Freeze image encoder, train prompt + mask decoderKeep SAM's generalist ability
Full fine-tune所有參數可訓練資料 > 10k 張，且 GPU 強All params trainable> 10k labels, strong GPU

五、實作範例

# Micro-SAM — interactive + automatic segmentation
from micro_sam import util
from micro_sam.instance_segmentation import AutomaticMaskGenerator

# Pick the right model
predictor = util.get_sam_model(model_type="vit_b_lm")   # LM = light microscopy
# Options: vit_b_lm, vit_l_lm, vit_h_lm, vit_b_em, vit_b_histopathology, ...

# Compute embedding once (cached)
embedding = util.precompute_image_embeddings(predictor, image)

# 1) Interactive with point prompt
from micro_sam.prompt_based_segmentation import segment_from_points
points = [[250, 300], [200, 290]]  # (y, x)
labels_pt = [1, 1]                          # 1 = foreground
mask = segment_from_points(predictor, points, labels_pt, embedding)

# 2) Automatic: scatter prompts and get all objects
amg = AutomaticMaskGenerator(predictor)
amg.initialize(image, embedding)
masks = amg.generate()

# 3) Volumetric / tracking — see micro_sam.multi_dimensional_segmentation
# MedSAM2 — 3D / video promptable segmentation
from medsam2 import build_medsam2_predictor

predictor = build_medsam2_predictor("medsam2_hiera_b.pt", device="cuda")
predictor.set_image(volume_or_video)   # 3D volume or T frames

# Provide a 2D click/box on one slice → propagated through volume/time
mask3d = predictor.predict_3d(
    point=[256, 256], slice_idx=20,
    propagate=True
)
# UNI — pathology patch embedding for downstream classification
import timm
import torch
from torchvision import transforms

# Load from HuggingFace hub (uni-pretrained ViT-L)

model.eval().cuda()
