高內涵分析

藥物 / 擾動 / Cell Painting 篩選一次產生 10⁴–10⁶ 張影像，組織成 plate × well × site × channel。挑戰從「分割這張」變成「在大規模上建可重現 pipeline、跨 plate 正規化、做表型 profile」。

HCS produces 10⁴–10⁶ images per experiment, organised as plate × well × site × channel. The challenge shifts from "segment this image" to "build a reproducible pipeline, normalize across plates, and extract phenotype profiles at scale."

HCS 架構

一、HCS 資料層次

典型 96 / 384 well plate：

  • Plate（一塊培養盤）→ 多個 Well（如 A1, A2 …）→ 多個 Site / Field（每 well 拍幾個位置）→ 多個 Channel（DAPI、Mito、ER、Actin、RNA、Golgi/PM）。
  • 每張影像必須對應 plate map：哪個 well 是 control、哪個是哪個化合物、濃度、replicate ID。
  • 每個物件還要記 per-cell features → 然後聚合到 well 層級（per-well profile）。

Typical 96/384-well plate:

  • Plate → multiple Wells (A1, A2, …) → multiple Sites/Fields per well → multiple Channels (DAPI, Mito, ER, Actin, RNA, Golgi/PM).
  • Each image must map to a plate map: which well is control, which compound at which dose, replicate ID.
  • Each cell yields per-cell features → aggregated to per-well profile.

Cell Painting

三、Cell Painting

Cell Painting 是非靶向 (unbiased) 的形態學表型 assay。一次染 6 條螢光染料 (5 個通道) 覆蓋細胞主要區室：

  • Hoechst → 核 DNA
  • SYTO 14 → 核仁 + 細胞質 RNA
  • Concanavalin A → ER
  • WGA → Golgi / 細胞膜
  • Phalloidin → F-actin (細胞骨架)
  • MitoTracker → 粒線體

從這 5 個通道萃取數百個 per-cell 特徵 → 聚合成 well-level morphological profile。同樣表型的擾動會有相似 profile (cosine similarity)。

JUMP-CP 公開了 3M+ 影像、12k+ 化合物 + 8k+ 基因擾動的世界級資源（Chandrasekaran 2024）。

Cell Painting is an unbiased morphological phenotyping assay. Six dyes (5 channels) cover the major cellular compartments:

  • Hoechst → nuclear DNA
  • SYTO 14 → nucleoli + cytoplasmic RNA
  • Concanavalin A → ER
  • WGA → Golgi / plasma membrane
  • Phalloidin → F-actin
  • MitoTracker → mitochondria

Hundreds of per-cell features → aggregated into a well-level morphological profile. Perturbations with similar phenotypes have similar profiles (cosine similarity).

JUMP-CP released 3M+ images of 12k+ compounds + 8k+ genetic perturbations (Chandrasekaran 2024).

正規化

四、Plate 效應與正規化

HCS 最大的陷阱是 plate effect：

  • Edge effect：外圈 well 蒸發 / 溫度不均，數據偏離
  • Plate-to-plate variation：染色強度、顯微鏡 drift
  • Row/column gradient：機械手液移誤差

標準流程：

The biggest trap in HCS is plate effects:

  • Edge effect: outer wells dry out or run hot
  • Plate-to-plate variation: staining intensity, microscope drift
  • Row/column gradient: liquid handler errors

Standard workflow:

QC 過濾

剔除 out-of-focus / 細胞太少 / 顯微鏡飄移的 site。每 well 細胞數應有最低門檻。

Drop out-of-focus / under-populated / drifted sites. Enforce a minimum cell count per well.

每細胞 → 每井聚合

用 median（不是 mean，CP 分佈重尾）+ MAD 聚合到 per-well profile。

Use median (not mean — distributions are heavy-tailed) and MAD per well.

每板正規化

每 plate 用 negative control 的 median ± MAD 做 robust z-score。或用 Typical Variation Normalization (TVN)、Sphering 校正 batch。

Robust z-score per plate using negative-control median ± MAD. Or TVN / sphering for batch correction.

特徵選擇

用 pycytominer 過濾相關性 > 0.95、low-variance、NaN 過多的特徵。

Use pycytominer to drop r > 0.95, low-variance, or NaN-heavy features.

表型配對

計算percent-replicating（replicate 之間的相似度勝過隨機的比例）；percent-matching（不同擾動有相同表型的比例）。論文 reporting 標準指標。

Compute percent-replicating (replicate similarity beating random) and percent-matching (cross-perturbation similarity). Standard reporting metrics.

import pandas as pd
from pycytominer import normalize, feature_select, aggregate

# Load per-cell features (CellProfiler output, parquet)
df = pd.read_parquet("single_cell_features.parquet")

# 1) Aggregate to per-well
profiles = aggregate(df,
    strata=["Metadata_Plate", "Metadata_Well"],
    features="infer", operation="median")

# 2) Plate-wise normalize against DMSO controls
norm = normalize(profiles,
    features="infer",
    meta_features=["Metadata_Plate","Metadata_Well","Metadata_perturbation"],
    samples="Metadata_perturbation == 'DMSO'",
    method="mad_robustize")

# 3) Feature select
sel = feature_select(norm, features="infer",
    operation=["variance_threshold", "correlation_threshold",
               "drop_na_columns", "blocklist"])

# 4) Percent replicating (simple version)
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np
feat_cols = [c for c in sel.columns if c.startswith(("Cells_","Nuclei_","Cytoplasm_"))]
X = sel[feat_cols].values
sim = cosine_similarity(X)
# Compare replicate sim vs. random null...
# CellProfiler in headless / batch mode (cluster-ready)
cellprofiler --run-headless \
    --pipeline analysis.cppipe \
    --image-directory /data/plate1/images \
    --output-directory /results/plate1 \
    --plugins-directory /opt/cp-plugins \

公式拆解

Robust z-score 與 percent-replicating

Σ 逐步推導
zi = (xi − medianDMSO) / MADDMSO
① per-plate normalizationper-plate normalization 每 plate 獨立計算 median 與 MAD；DMSO controls 為 baseline。Compute median and MAD per plate; DMSO controls are the baseline.
② robust 抗 outlierRobust to outliers median + MAD 比 mean + std 抗極端值，CP heavy-tailed 必用。Median + MAD beats mean + std on heavy-tailed CP distributions.
③ cosine similarityCosine similarity Profile A 與 B 的「角度相似度」：cos(A,B) = A·B / (|A||B|)。Profile A vs B 'angular similarity': cos(A,B) = A·B / (|A||B|).
④ Percent-replicatingPercent-replicating #(replicate pair 相似度 > 95% null) / total — 衡量 assay reproducibility。#(replicate pairs above the 95% null) / total — measures assay reproducibility.
推論：plate-wise z-score 是 HCS pipeline 的核心。沒做正規化的 raw value 不能跨 plate 比較。Percent-replicating 60–80% 為良好 assay；<50% 通常需重新檢視。
Implication: per-plate z-score is the core of every HCS pipeline. Raw values without it can't be compared across plates. Percent-replicating 60–80% is a good assay; <50% usually needs a rework.
思辨討論

HCS 三大爭論

A. Edge effect 比 row/column gradient 危險？
兩者都是 plate 效應，差異在哪？
Both are plate effects — what differs?
提示：edge 涉及蒸發 / 溫度差，難用 linear regression 模型；gradient 通常是液移誤差，較易校正。
Hint: edges involve evaporation / temperature — hard to model linearly; gradients are liquid-handler errors and easier to correct.
B. Cell Painting 真的「unbiased」嗎？
5 個染料怎麼選的？是否真的看到所有 phenotype？
How were the 5 dyes chosen? Does it really capture all phenotypes?
提示：Cell Painting 染料是經驗選擇，涵蓋核 / RNA / ER / Golgi / Actin / Mito。沒染到的 phenotype（如代謝產物）看不到。
Hint: the dye set is empirical, covering nucleus / RNA / ER / Golgi / Actin / Mito. Phenotypes outside that (e.g., metabolites) won't be captured.
C. DeepProfiler 取代 CP features 的 trade-off？
UNI embedding 比 1,700 CP features 強，但可解釋性差？
UNI embeddings beat 1,700 CP features but lose interpretability?
提示：UNI 性能高但黑箱；CP 每個 feature 都可解釋（area, intensity, texture）。生產用 UNI、論文圖 CP，是常見折衷。
Hint: UNI is high-performance but black-box; CP features are each interpretable (area, intensity, texture). Use UNI for production and CP for figures — a common compromise.
