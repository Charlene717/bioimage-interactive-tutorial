STEP 8 / 9

細胞 / 粒子追蹤

動態生物（分裂、遷移、訊號振盪、譜系關係）需要把分割物件跨時間連起來。追蹤錯誤會累積——第 30 幀的 ID switch 會毀掉之後整條譜系。

Dynamic biology (division, migration, signalling oscillations, lineage) requires linking segmented objects across time. Tracking errors compound — an ID switch at frame 30 corrupts the rest of the lineage.

開場故事 · STORY

從 1965 Hungarian 算法到 2024 Transformer 追蹤

1996 年 John Crocker 與 Eric Weeks 在膠體物理發明了 nearest-neighbour particle tracking (TrackPy 前身)；2008 年 Khuloud Jaqaman 在 Nat Methods 把 1965 Kuhn-Munkres 的 Linear Assignment Problem (LAP) 帶入 cell tracking。

2024 年 Trackastra Transformer 拿下 Cell Tracking Challenge 冠軍——但它依賴的數學基礎仍是匈牙利算法。從追蹤一個分子在膜上的擴散，到追蹤胚胎中每個細胞 5 代分裂，「跨時間連起來」是動態生物學唯一的入口。

In 1996 John Crocker and Eric Weeks invented nearest-neighbour particle tracking in colloidal physics (the predecessor of TrackPy); in 2008 Khuloud Jaqaman brought the Linear Assignment Problem (LAP) from Kuhn-Munkres (1965) into cell tracking via Nat Methods.

In 2024 Trackastra (Transformer-based) won the Cell Tracking Challenge — but its math still rests on the Hungarian algorithm. From a molecule diffusing on a membrane to every cell in an embryo across 5 division generations, 'linking across time' is the only doorway to dynamic biology.

兩大範式

一、兩大範式

🎯 Tracking-by-Detection

(1) 每一幀獨立做分割 / 偵測；
(2) 用 linker（LAP、Kalman、graph match）把跨幀物件連起來。
大多數工具屬此類：TrackMate、btrack、ilastik。優點：模組化、可換 segmenter。

(1) Per-frame segmentation/detection;
(2) A linker (LAP/Kalman/graph) connects objects across frames.
Most tools follow this: TrackMate, btrack, ilastik. Pros: modular, swap segmenter freely.

🌐 聯合偵測追蹤

單一 DL 模型同時輸出物件 + 跨幀關聯。代表：Trackastra（transformer pairwise association、2024 CTC 冠軍）。優點：對相黏 / 快速移動更穩。

A single DL model jointly outputs detections + cross-frame associations. Example: Trackastra (transformer pairwise association, winner of CTC 2024). Pros: more robust on touching / fast-moving objects.

評估

三、追蹤評估指標

  • MOTA (Multi-Object Tracking Accuracy)：綜合 FN、FP、ID switch
  • IDF1：基於 ID 對齊的 F1，對 ID 切換更敏感
  • Cell Tracking Challenge 三大指標：
    　DET：每幀偵測準確度
    　SEG：分割 IoU
    　TRA：追蹤拓撲（節點 + 邊）正確性
  • Mitosis F1：分裂事件偵測 F1，譜系研究必看
  • MOTA: combined FN, FP, ID switches
  • IDF1: ID-alignment-based F1, more sensitive to ID switches
  • Cell Tracking Challenge metrics:
    　DET: per-frame detection accuracy
    　SEG: segmentation IoU
    　TRA: tracking topology (nodes + edges) correctness
  • Mitosis F1: division-event detection F1 — essential for lineage studies
🚫
原則：先優化分割，再調追蹤參數。5% 的偵測漏失常造成 30% 的 ID switch。在分割還沒到位之前，動 tracker 是徒勞。 Rule: optimize segmentation first, tracker parameters second. A 5% detection miss typically causes ~30% ID switches. Tuning the tracker before segmentation is solid is wasted effort.

// TrackMate 7 — Cellpose detector + LAP linker
// 1) Plugins > Tracking > TrackMate
// 2) Detector: Cellpose detector  →  Custom model / cyto3
// 3) Initial thresholding on quality
// 4) Tracker: Simple LAP tracker
//    - Frame-to-frame max distance: ~15 px
//    - Gap-closing max distance: ~25 px, max frame gap: 2
//    - Splitting/merging: ON if you have divisions
// 5) Save XML → reusable as input to other tools

// Or call via Jython for automation
from fiji.plugin.trackmate import Settings, TrackMate, Model
settings = Settings(imp)
settings.detectorFactory = CellposeDetectorFactory()
settings.trackerFactory  = SimpleSparseLAPTrackerFactory()
tm = TrackMate(model, settings)
tm.process()
# btrack — Bayesian tracking with lineage
import btrack
from btrack import datasets

# Convert segmentation labels to btrack objects
objects = btrack.utils.segmentation_to_objects(labels_stack)

with btrack.BayesianTracker() as tracker:
    tracker.configure("cell_config.json")
    tracker.append(objects)
    tracker.volume = ((0, 1200), (0, 1600))   # (xlim, ylim)
    tracker.track(step_size=100)
    tracker.optimize()
    tracks = tracker.tracks       # list of Track objects with ID, t, x, y, parent

# Export as napari layer
data, properties, graph = tracker.to_napari()
# In napari: viewer.add_tracks(data, properties=properties, graph=graph)
# Trackastra — transformer-based tracker
from trackastra.model import Trackastra
from trackastra.tracking import graph_to_napari_tracks

# Load pretrained general model
model = Trackastra.from_pretrained("general_2d", device="cuda")

# Input: image stack + corresponding label stack (instance segmentation per frame)
track_graph = model.track(imgs, masks, mode="greedy")

# Convert to napari tracks
data, properties, graph = graph_to_napari_tracks(track_graph)

# Or export as CSV / CTC format
from trackastra.tracking import ctc_tools
ctc_tools.write_ctc_data(track_graph, out_dir="results/")

公式拆解

LAP 最佳配對

Σ 逐步推導
minimize Σij cij xij s.t. Σj xij = 1, Σi xij = 1, xij ∈ {0,1}
① cost 矩陣Cost matrix c_ij = 第 t 幀第 i 細胞與第 t+1 幀第 j 細胞的「成本」（歐氏距離 + intensity diff）。c_ij = the 'cost' of matching cell i in frame t to cell j in frame t+1 (Euclidean distance + intensity diff).
② 約束Constraints 每個細胞最多配 1 個下家；每個下家最多收 1 個上家。Each cell links to at most one next; each next receives at most one predecessor.
③ Hungarian 解Hungarian solution O(n³) 解出全域最佳配對；遠優於 nearest-neighbour 的 greedy。O(n³) solves the global optimum; far better than greedy nearest-neighbour.
④ Gap closing 擴展Gap closing extension 容許跨多幀的「擴展節點」，把短暫漏偵測接回。Allow multi-frame 'extended nodes' to reconnect short detection gaps.
推論：5% detection FN 為何爆 30% ID switch — 每個漏偵測都讓 t-1 與 t+1 之間找不到對應，被迫各自起新軌跡。Gap closing 可以救一半，但根源仍是 detection。
Implication: a 5% detection FN explodes into 30% ID switches because every missed detection breaks the t-1 ↔ t+1 link, forcing both sides to start new tracks. Gap closing rescues some, but the root cause is detection.

精選案例

兩個追蹤領域的傳奇

① TrackMate 7：Fiji 上的整合 tracker
Ershov et al., Nat Methods 19:829 (2022) · imagej.net/plugins/trackmate

Jean-Yves Tinevez (Institut Pasteur) 等 2017 在 Methods 推出 TrackMate；2022 TrackMate 7 在 Nat Methods 升級：整合 Cellpose、StarDist、Weka classifier 為 detector，並保留原有 GUI。一個 Fiji plugin 從 detector 到 LAP linker 到 lineage 視覺化全部搞定。Fiji 教學首選的同時，也是 reproducible：所有設定可存成 XML 給下次或審稿。

Jean-Yves Tinevez (Institut Pasteur) released TrackMate (Methods 2017); TrackMate 7 in Nat Methods 2022 upgraded it: integrating Cellpose, StarDist, and Weka classifiers as detectors while keeping the GUI. A single Fiji plugin handles detector → LAP linker → lineage visualization. The Fiji teaching default — and reproducible: settings save as XML for reuse or peer review.

啟示：好的工具讓互動式探索與可重現分析「無縫接軌」。
Lesson: a great tool makes interactive exploration and reproducible analysis 'seamless.'
② Trackastra：Transformer 重塑 cell tracking
Gallusser & Weigert, ECCV 2024 · 2024 Cell Tracking Challenge winner

Benjamin Gallusser 與 Martin Weigert 在 EPFL 發表 Trackastra：用 Transformer 學 pair-wise association，端到端訓練。在 2024 Cell Tracking Challenge 第七屆奪冠，特別在「相黏細胞」與「快速移動」情境刷掉所有傳統 LAP 方法。Trackastra 也率先公開 napari plugin，讓使用者直接拖放 dataset。

Benjamin Gallusser and Martin Weigert (EPFL) published Trackastra: a Transformer learning pairwise associations, end-to-end trained. It won the 7th Cell Tracking Challenge in 2024, beating all classical LAP methods on touching cells and fast motion. Trackastra also shipped an early napari plugin for drag-and-drop datasets.

啟示：transformer 不只是 LLM 的工具，序列關聯本就是它的強項——追蹤=時間序列關聯，完美吻合。
Lesson: transformers aren't just for LLMs — sequence association is their strength, and tracking = temporal association, a perfect fit.
