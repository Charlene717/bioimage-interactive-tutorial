AI 基礎模型

在數十億張自然影像上預訓練的基礎模型，能以「提示 (prompt)」方式分割顯微影像，零訓練橫跨多個術式——但原版 SAM 在生物上很少是最佳選擇。

Foundation models pretrained on billions of natural images segment microscopy via prompts — zero-shot across modalities. But raw SAM is rarely the right model for biology on its own.

開場故事 · STORY

從 SAM 到 Micro-SAM：兩年內變天的 bioimage AI

2023 年 4 月 Meta AI 發表 Segment Anything (SAM)：「任何影像、任何 prompt、零訓練分割」。3 個月內 napari、QuPath、Fiji 都長出 SAM plugin。但研究者很快發現：SAM 在 EM 切片上慘不忍睹，在低 SNR fluorescence 也力不從心。

2025 年 2 月 Anwai Archit 等在 Nat Methods 發表 Micro-SAM (μSAM)，把 SAM fine-tune 到光學+電子顯微全頻譜；同年 MedSAM2、CellSAM、UNI 2 接連登場。Bioimage 進入「prompt-first」時代——標一張就能跑全 dataset。

In April 2023 Meta AI released Segment Anything (SAM): 'any image, any prompt, zero-shot segmentation.' Within 3 months napari, QuPath, and Fiji all sprouted SAM plugins. But researchers quickly noticed: SAM was terrible on EM sections and struggled on low-SNR fluorescence.

In February 2025 Anwai Archit et al. published Micro-SAM (μSAM) in Nat Methods, fine-tuning SAM for the full light + EM microscopy spectrum; the same year brought MedSAM2, CellSAM, and UNI 2. Bioimage entered the 'prompt-first' era — annotate one, run on the whole dataset.

觀念

一、提示式分割

傳統 DL 分割：訓練好的模型輸入影像，輸出 mask，無法互動。

SAM (Segment Anything Model, 2023) 開啟了新範式——用戶給提示 (prompt)：點一下、畫個 box、給張粗略 mask、甚至文字，模型即時回傳該物件的精確 mask。

四種主要 prompt 類型：

  • Point：點擊「我要這個物件」
  • Box：拉個 bounding box
  • Mask：粗略 mask → 精細化
  • Text：「all cells」「nuclei only」（CLIP 結合）

Traditional DL: a trained model takes an image and outputs a mask — not interactive.

SAM (Segment Anything Model, 2023) opened a new paradigm — the user provides a prompt (a point, box, rough mask, or text) and the model returns a precise mask in real time.

Four main prompt types:

  • Point: "I want this object"
  • Box: a bounding box
  • Mask: coarse mask → refined
  • Text: "all cells", "nuclei only" (CLIP-grounded)

工作流程

三、互動式分割流程

開啟 napari + Micro-SAM

pip install micro-sam napari → napari 啟動，安裝 plugin。

pip install micro-sam napari → run napari, install the plugin.

計算 embedding

選好模型（vit_b_lm, vit_b_em, vit_h_lm…），SAM encoder 跑一次得到嵌入向量。之後所有 prompt 都即時。

Pick a model (vit_b_lm, vit_b_em, vit_h_lm…). SAM encoder runs once to compute embeddings; all prompts thereafter are real-time.

互動 prompt

切到 napari 的 point / box layer，點一下細胞 → 即時 mask；不夠精細可再點 negative point 排除背景。

Use napari's point/box layer — click a cell to get an instant mask; refine with negative points to exclude background.

自動模式

AMG = Automatic Mask Generation：在整張影像撒點，過濾低品質結果，得到所有物件 mask。可作為其他模型的 pseudo-label。

AMG = Automatic Mask Generation: scatter point prompts across the image, filter low-quality outputs to get all object masks. Useful as pseudo-labels for other models.

3D / 影片

Micro-SAM 提供 volumetric 與 tracking 模式：在 2D 標一張，跨 z 或 t 自動追蹤連結。

Micro-SAM offers volumetric & tracking modes: label one 2D slice, then auto-propagate across z or t.

⚠️
不要把 SAM 用在 3D：原版 SAM 是 2D 模型，逐切片獨立跑會造成 z 方向 ID 不一致。請改用 Micro-SAM volumetric / MedSAM2 / Cellpose 3D。 Don't run vanilla SAM on 3D: it's a 2D model; per-slice independent runs produce inconsistent IDs across z. Use Micro-SAM volumetric / MedSAM2 / Cellpose 3D instead.

from micro_sam import util
from micro_sam.instance_segmentation import AutomaticMaskGenerator

# Pick the right model
predictor = util.get_sam_model(model_type="vit_b_lm")   # LM = light microscopy
# Options: vit_b_lm, vit_l_lm, vit_h_lm, vit_b_em, vit_b_histopathology, ...

# Compute embedding once (cached)
embedding = util.precompute_image_embeddings(predictor, image)

# 1) Interactive with point prompt
from micro_sam.prompt_based_segmentation import segment_from_points
points = [[250, 300], [200, 290]]  # (y, x)
labels_pt = [1, 1]                          # 1 = foreground
mask = segment_from_points(predictor, points, labels_pt, embedding)

# 2) Automatic: scatter prompts and get all objects
amg = AutomaticMaskGenerator(predictor)
amg.initialize(image, embedding)
masks = amg.generate()

# 3) Volumetric / tracking — see micro_sam.multi_dimensional_segmentation
# MedSAM2 — 3D / video promptable segmentation
from medsam2 import build_medsam2_predictor

predictor = build_medsam2_predictor("medsam2_hiera_b.pt", device="cuda")
predictor.set_image(volume_or_video)   # 3D volume or T frames

# Provide a 2D click/box on one slice → propagated through volume/time
mask3d = predictor.predict_3d(
    point=[256, 256], slice_idx=20,
    propagate=True
)

SAM 的數學骨架

Σ 逐步推導
mask = MaskDecoder( ImageEncoder(image), PromptEncoder(prompt) )

② Prompt encodingPrompt encoding Point / box / mask / text 各有獨立 encoder，輸出 prompt embedding。Point / box / mask / text each have an encoder producing a prompt embedding.
③ Cross attentionCross attention MaskDecoder 用 cross-attention 把 image + prompt embedding 融合。MaskDecoder fuses image + prompt embedding via cross-attention.
④ Mask 輸出Mask output 上採樣到 1024×1024，加上 IoU prediction head 給 confidence。Upsamples to 1024×1024, with an IoU prediction head for confidence.

精選案例

兩個改寫 bioimage AI 的故事

① Micro-SAM：在 napari 內 5 分鐘標 100 細胞
Archit et al., Nat Methods 22:579 (2025) · github.com/computational-cell-analytics/micro-sam

Anwai Archit 等在 Göttingen 為了讓 SAM 在 EM 與低 SNR fluorescence 真正可用，把 SAM 微調到 30+ 顯微 modality。他們的 napari plugin 工作流：算一次 embedding → 點 cell → 即時 mask → 點 negative point 排除背景 → 5 分鐘 100 細胞。整個 Göttingen 細胞生物實驗室一年內捨棄手動標註，全部 prompt-first。

Anwai Archit et al. (Göttingen) made SAM actually usable on EM and low-SNR fluorescence by fine-tuning across 30+ microscopy modalities. Their napari plugin: one embedding → click a cell → instant mask → click negative points to exclude background → 100 cells in 5 minutes. The Göttingen cell-biology institute abandoned manual annotation entirely within one year.

啟示：foundation model 真正落地的「最後一哩」是領域適配 + 良好 UI。
Lesson: the 'last mile' of foundation models in practice is domain adaptation + good UI.
② UNI：100M patch 訓出的病理通用嵌入
Chen et al., Nat Medicine 30:850 (2024) · Mahmood Lab DFCI

Richard Chen 等在 Dana-Farber Mahmood Lab 用 DINOv2 訓練 UNI：~100M 病理 patches、ViT-L backbone、自監督預訓練。在 ~30 個下游任務（cancer 分類、生物標誌預測、prognosis 預測）平均超越 ImageNet pretrained baseline 約 20%。UNI 2 (2025) 進一步擴展到 200k slides，重塑整個 computational pathology 領域。

Richard Chen et al. (DFCI Mahmood Lab) trained UNI with DINOv2: ~100M pathology patches, ViT-L backbone, self-supervised. On ~30 downstream tasks (cancer classification, biomarker prediction, prognosis), it beats ImageNet baselines by ~20% on average. UNI 2 (2025) extended to 200k slides, reshaping computational pathology.

啟示：在病理這種「特殊視覺領域」，自訓 foundation model 仍有顯著價值。
Lesson: in 'specialized visual domains' like pathology, training a dedicated foundation model still pays dividends.

In under 24 months the bioimage community fully domain-adapted general-purpose AI. A classic example of scientific 'relay sprinting' — and proof that open source + good interfaces win.
