STEP 9 / 9

量化與特徵

分割不是終點，量化才是。把每個物件變成一個「影像表型向量」——強度、形態、共定位、紋理——這就是 Cell Painting、表型分群、統計檢定的原料。

Segmentation isn't the goal — quantification is. Each object becomes an "image phenotype vector" (intensity, morphology, colocalization, texture) — the raw material for Cell Painting, phenotyping, and statistics.

特徵家族

一、特徵家族

💡 強度特徵

mean / median / std / CV / p10–p90 / integrated density。注意：要不要扣背景？背景如何定義（cell-free region、邊緣 ring、global）？

mean / median / std / CV / percentiles / integrated density. Caveats: subtract background? How is background defined (cell-free region, edge ring, global)?

🔷 形態特徵

area / perimeter / eccentricity / solidity / Feret diameter / major-minor axis / Zernike moments。3D 加 surface area / volume / sphericity。

area / perimeter / eccentricity / solidity / Feret diameter / axis lengths / Zernike moments. 3D adds surface area, volume, sphericity.

🎨 紋理

Haralick GLCM (contrast/correlation/energy/homogeneity)、LBP、Gabor、Granularity、wavelet。對亮度級數量化（如 32 級）很敏感。

Haralick GLCM (contrast/correlation/energy/homogeneity), LBP, Gabor, Granularity, wavelet. Highly sensitive to grey-level quantization (e.g., 32 levels).

🤝 共定位

Pearson r、Manders M1/M2、Costes 隨機化檢定、object-based colocalization。背景未扣除會嚴重扭曲 Pearson r。

Pearson r, Manders M1/M2, Costes randomization, object-based colocalization. Pearson is heavily biased by uncorrected background.

🌐 空間統計

nearest-neighbour 距離、Ripley's K、density、neighbourhood composition。多重染色組織、spatial omics 必備。

Nearest-neighbour distance, Ripley's K, density, neighbourhood composition. Essential for multiplexed tissue / spatial omics.

🧬 Radiomics

PyRadiomics 提供 IBSI 標準特徵：shape、first-order、GLCM、GLRLM、GLSZM、NGTDM、wavelet。臨床 / 醫學影像常用。

PyRadiomics provides IBSI-standard features: shape, first-order, GLCM, GLRLM, GLSZM, NGTDM, wavelet. Common in clinical / medical imaging.

共定位深探

二、共定位陷阱

共定位看似「兩通道一起亮多少」，但細節決定論文是否可信。

  • Pearson r 對線性偏移敏感：兩通道同步背景升高，r 不變；獨自背景升一個，r 大降——所以必須做背景減除。
  • Manders 需閾值：M1 = (有 ch2 信號的 ch1 強度) / (全 ch1 強度)。閾值選不對結果飄移嚴重 → 用 Costes 自動閾值。
  • Costes 隨機化檢定：把一個通道區塊化隨機洗牌，看 r 在隨機分佈中的位置。p < 0.05 才算顯著高於隨機共定位。
  • Object-based：對胞器級結構（粒線體、囊泡），先分割再算「重疊比例」比像素級更合理。

Colocalization looks like "how much do two channels co-light?" but the details decide whether the paper is trustworthy.

  • Pearson r is sensitive to linear offset: simultaneous background rise leaves r unchanged; rise in just one channel drops r dramatically → background-subtract first.
  • Manders requires a threshold: M1 = (ch1 intensity where ch2 is positive) / (total ch1). Bad threshold → unstable result → use Costes auto-threshold.
  • Costes randomization: shuffle blocks of one channel and see where r sits in the null distribution. Only p < 0.05 vs. random counts as significant.
  • Object-based: for organelle-level structures (mitochondria, vesicles), segment first then compute fractional overlap — more meaningful than pixel-level metrics.

How many conditions / cells? → < 100 cells → boxplots; > 1000 → UMAP + clustering
Q2:
Across plates / batches? → Per-plate z-score / robust z (per-well median, not global mean)
Q3:
Highly correlated features? → pycytominer feature_select to drop r > 0.95 and near-zero-variance features

# Per-cell Haralick texture (loop over labels)
def haralick_per_obj(img, labels, levels=32):
    img_q = (img.astype("float32")/img.max()*(levels-1)).astype("uint8")
    feats = []
    for lbl in np.unique(labels)[1:]:
        ys, xs = np.where(labels==lbl)
        if len(ys)<10: continue
        patch = img_q[ys.min():ys.max()+1, xs.min():xs.max()+1]
        glcm = graycomatrix(patch, [1], [0,np.pi/4,np.pi/2], levels=levels, symmetric=True, normed=True)
        feats.append({"label":lbl,
            "contrast":graycoprops(glcm,"contrast").mean(),
            "homog":graycoprops(glcm,"homogeneity").mean(),
            "correlation":graycoprops(glcm,"correlation").mean()})
    return pd.DataFrame(feats)

# Manders with Costes auto-threshold
def costes_thresholds(c1, c2):
    flat1, flat2 = c1.ravel(), c2.ravel()
    # Orthogonal regression line through origin
    a = np.cov(flat1, flat2)[0,1] / np.var(flat1)
    b = flat2.mean() - a*flat1.mean()
    # Iterate down until Pearson on (T1,T2) area drops to ~0
    T1 = flat1.max()
    while T1 > 0:
        T2 = a*T1 + b
        mask = (flat1 < T1) | (flat2 < T2)
        if mask.sum() < 100: break
        rr, _ = pearsonr(flat1[mask], flat2[mask])
        if rr <= 0.01: break
        T1 -= 1
    return T1, T2
