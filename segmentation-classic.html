閾值

一、閾值法

把灰階影像變成二值前景 / 背景，是最基本的分割。重點是怎麼自動決定那條線。

Convert grayscale to foreground/background — the most basic segmentation. The key question is how to choose that line automatically.

形態學

二、形態學運算

在二值或灰階影像上做的局部鄰域運算，是清理閾值結果的核心工具。

  • Erosion 侵蝕：縮小前景、斷開細小連接
  • Dilation 膨脹：擴大前景、填洞
  • Opening = erosion + dilation：去掉小亮點（雜訊）
  • Closing = dilation + erosion：填補小洞、連接斷裂物件
  • Top-hat / Bottom-hat：突顯小亮 / 小暗結構（如 spot detection、puncta）
  • Reconstruction：保留與 marker 連通的整個前景，常用於 hole filling

Local neighbourhood operations on binary or grayscale images — the workhorse for cleaning up thresholded masks.

  • Erosion: shrinks foreground, breaks thin connections
  • Dilation: enlarges foreground, fills holes
  • Opening = erosion + dilation: removes small bright specks
  • Closing = dilation + erosion: fills small holes, joins broken objects
  • Top-hat / Bottom-hat: highlights small bright/dark structures (spot detection, puncta)
  • Reconstruction: keeps connected regions seeded by markers — used for hole filling
Watershed

三、Watershed 分離相黏物件

把灰階影像想成地形（亮 = 高、暗 = 低），淹水的時候，不同集水盆地的邊界就是分割線。對相黏細胞 / 核非常有用。

實作步驟（marker-controlled watershed）：

Treat the grayscale image as terrain (bright = high, dark = low). As water rises, boundaries between catchment basins become segmentation lines — invaluable for touching cells/nuclei.

Practical workflow (marker-controlled watershed):

閾值得到 mask

距離變換

每個前景像素的值 = 到最近背景的距離。細胞中心會是局部最大值。

Each foreground pixel = distance to nearest background. Cell centres become local maxima.

找 markers

用 peak_local_max 找出每個物件的 marker。min_distance 是關鍵：太小 → 過分割，太大 → 漏分割。

Use peak_local_max to pick one marker per object. min_distance is the key knob: too small → over-segmentation, too large → under-segmentation.

跑 watershed

從 markers 灌水，每個 marker 的水域就是一個 cell。

Flood from markers; each basin = one cell.

⚠️
Watershed 經典陷阱：過分割（一個細胞被切成多塊）。解法：(1) 先 Gaussian smooth 距離變換；(2) 用 h-minima 去除淺局部最大值；(3) 後處理時合併小於閾值的相鄰區。 Classic pitfall: over-segmentation (one cell sliced into many). Fixes: (1) Gaussian-smooth the distance map; (2) suppress shallow maxima via h-minima; (3) post-merge adjacent tiny regions.
程式碼

四、完整 pipeline

import numpy as np
from skimage import io, filters, morphology, measure, segmentation
from scipy import ndimage as ndi

img = io.imread("nuclei.tif").astype("float32")

# 1) Pick a threshold
filters.try_all_threshold(img)                  # Visual comparison
t = filters.threshold_otsu(img)
mask = img > t

# 2) Morphological cleanup
mask = morphology.opening(mask, morphology.disk(2))    # remove specks
mask = ndi.binary_fill_holes(mask)                              # fill holes
mask = morphology.remove_small_objects(mask, min_size=100)

# 3) Distance transform + markers
dist = ndi.distance_transform_edt(mask)
local_max = morphology.local_maxima(filters.gaussian(dist, 2))
markers, _ = ndi.label(local_max)

# 4) Watershed
labels = segmentation.watershed(-dist, markers, mask=mask)

# 5) Measure
table = measure.regionprops_table(labels, intensity_image=img,
    properties=["label","area","eccentricity","mean_intensity","centroid"])
import pandas as pd
df = pd.DataFrame(table)
print(df.head())
// Fiji classical nucleus segmentation
run("Bio-Formats Importer", "open=[/path/nuclei.tif]");

// 1) Smooth and threshold
run("Gaussian Blur...", "sigma=1");
setAutoThreshold("Otsu dark");
run("Convert to Mask");

// 2) Morphological cleanup (built-in)
run("Open");
run("Fill Holes");

// 3) Watershed (MorphoLibJ has better one)
run("Watershed");                              // quick built-in
// Or: Plugins > MorphoLibJ > Marker-controlled watershed

// 4) Measure
run("Set Measurements...", "area mean centroid display redirect=None decimal=3");
run("Analyze Particles...", "size=100-Infinity show=[Count Masks] display");

公式拆解

Otsu 公式拆解

Σ 逐步推導
σ²B(t) = ω0(t) · ω1(t) · [μ0(t) − μ1(t)]²
① 類別權重Class weights ω₀(t) = 閾值 t 以下像素比例；ω₁(t) = 1 − ω₀。ω₀(t) = fraction of pixels ≤ t; ω₁(t) = 1 − ω₀.
② 類別均值Class means μ₀(t) = 閾值以下像素平均；μ₁(t) = 閾值以上平均。μ₀(t) = mean of pixels ≤ t; μ₁(t) = mean of pixels > t.
③ 類間變異Between-class variance σ²_B 衡量兩類「差異有多大」乘上「兩類各佔多少」。σ²_B measures 'how different the two classes are' weighted by their sizes.
④ 最佳閾值Optimal threshold t* = argmax_t σ²_B(t)。對 256 級灰階就掃 256 次，O(N) 線性。t* = argmax_t σ²_B(t). 256 grey levels = 256 scans; O(N) linear.
失效情境：直方圖近單峰（前景 <5%）時 Otsu 飄移；改用 Triangle 或 Li。Adaptive Otsu 可以滑窗局部執行。
Failure mode: nearly unimodal histogram (foreground <5%) makes Otsu drift; switch to Triangle or Li. Adaptive Otsu can be applied with a sliding window.

兩個改寫傳統分割的故事

① scikit-image：Python 影像處理的支柱
van der Walt et al., PeerJ 2:e453 (2014) · 14 年社群開源故事

Stéfan van der Walt 2009 在 Berkeley 寫了 scikit-image 第一版，2014 PeerJ 論文發表時已有 100+ 貢獻者。如今 scikit-image 是 Python 影像處理事實標準：regionprops / morphology / segmentation 模組支撐了上萬篇論文。0.24 (2025) 加入 GPU backend (cupy)，傳統流程仍在演進。

Stéfan van der Walt wrote scikit-image's first version at Berkeley in 2009. By the PeerJ paper (2014) it had 100+ contributors. It's now the Python image-processing standard: regionprops / morphology / segmentation modules underpin tens of thousands of papers. v0.24 (2025) added a GPU backend (cupy) — the classical pipeline keeps evolving.

啟示：好工具不需要 DL；社群協作的開源是更持久的基礎建設。
Lesson: good tools don't need DL; community open-source is more durable infrastructure.
② MorphoLibJ：3D 形態學的 Swiss Army knife
Legland et al., Bioinformatics 2016 · Fiji update site

David Legland 在 INRAE 寫的 MorphoLibJ 把 3D 形態學運算帶入 Fiji：3D opening / closing / watershed / region adjacency graph。一直到 2024 Cellpose 3D 出現前，這是 3D 共軛焦 nucleus segmentation 的最強傳統流程。Plant biology、organoid 研究尤其受惠。

David Legland's MorphoLibJ (INRAE) brought 3-D morphology to Fiji: 3-D opening / closing / watershed / region adjacency graph. Until Cellpose 3D (2024), it was the strongest classical pipeline for 3-D confocal nucleus segmentation — especially loved by plant biology and organoids.

啟示：在「DL 一切」之前，傳統流程在某些子問題上仍未被超越。
Lesson: before 'DL everything,' classical pipelines remain unbeaten on some sub-problems.
